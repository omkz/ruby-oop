## How to run the app

### interactive mode
`$ ./my_program`

### in the repl:

input:
`create_parking_lot 6`

input:
`park('KA­01­HH­9999', 'White')`

### show all of output in shell from file_inputs.txt
`$ ./my_program file_inputs.txt`

### writes all of output into output.txt
`$ ./my_program file_inputs.txt > output.txt`
